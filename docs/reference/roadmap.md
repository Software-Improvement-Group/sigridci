Sigrid Roadmap
==============

This page shows the high-level product roadmap for Sigrid. In practice, SIG develops Sigrid using an
[agile process](https://en.wikipedia.org/wiki/Agile_software_development) and
[continuous delivery](https://en.wikipedia.org/wiki/Continuous_delivery). Please check the
[release notes](release-notes.md) to keep track of the latest features and improvements.
