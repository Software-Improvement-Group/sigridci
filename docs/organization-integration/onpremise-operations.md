# Sigrid On-Premise: Operations

This documentation covers on-premise Sigrid. It is not applicable for cloud-based Sigrid.
{: .attention }

This section provides operational guidance for managing Sigrid On-Premise in a production environment. It covers the key procedures required to deploy, update, and maintain the platform using Helm.

The content is intended for platform and operations teams and focuses on practical, repeatable workflows to ensure a stable and maintainable Sigrid deployment.

## Prerequisites

- You have read the Sigrid On-Premise documentation, including deployment and analysis setup
- You have access to Software Improvement Group AWS ECR registry
